Prüfungen erstellen... bald!

Hier wird fleissig an einer Plattform gearbeitet, die das Erstellen, Verwalten und Teilen von Prüfungsaufgaben vereinfachen soll. Möchten Sie gerne informiert werden, wenn es los geht? Dann können Sie hier Ihre E-Mail Adresse angeben.

So funktioniert doxblox

Kombinieren

Für eine Prüfung (oder Arbeits­blatt) werden Aufgaben­blöcke neu zusammen­gestellt. Die Aufgaben­blöcke stammen entweder aus einer öffentlichen Biblio­thek oder aus der eigenen Sammlung.

Erstellen

Aus den kombinierten Aufgaben­blöcken wird auto­matisch eine Prüfung generiert. Diese lässt sich als Word-Dokument exportieren und aus­drucken.

Verwalten

Prüfungen können einer Klasse zu­ge­wiesen werden. Damit erhällt man einen Über­blick über durch­geführte Prüfungen und kann Aufgaben­blöcke daraus wieder für neue Prüfungen ver­wenden.
